Adverb and Its Kinds

An Adverb is a word that describes a verb, an adjective, or another adverb.

Example:
She runs quickly.

Kinds of Adverbs:

1. Manner: Describes how an action is done.
Example: quickly, slowly

2. Place: Describes where an action happens.
Example: here, there, everywhere

3. Time: Describes when an action happens.
Example: now, yesterday, soon

4. Frequency: Describes how often an action happens.
Example: always, often, never

5. Degree: Describes the intensity of an action or adjective.
Example: very, too, almost

Adverb (क्रिया विशेषण) वह शब्द है जो किसी क्रिया, विशेषण या अन्य क्रिया विशेषण का वर्णन करता है।

उदाहरण:
वह तेज़ दौड़ती है।

क्रिया विशेषण के प्रकार:

1. Manner (किस प्रकार):
उदाहरण: धीरे, जल्दी

2. Place (स्थान):
उदाहरण: यहाँ, वहाँ, हर जगह

3. Time (समय):
उदाहरण: अब, कल, जल्द

4. Frequency (बारंबारता):
उदाहरण: हमेशा, अक्सर, कभी नहीं

5. Degree (स्तर/मात्रा):
उदाहरण: बहुत, अधिक, लगभग
